timeCountDown

倒计时插件的使用比较简单,只要建立一个元素div或者span之类，然后把函数绑定在元素上即可。由于插件中已经写好了倒计时部分的html代码，我们在样式中规定好相应class的样式即可（.ct-day、.ct-hour等等）。
然后我们要在参数中规定好开始时间和结束时间（{start: new Date("August 8, 2011 23:59:59"),end: new Date("August 12, 2011 23:59:59")}），或者用UTC时间也是可以的（start: 1313064000000,end: 1313071200000）。

在实际应用中，只要确定结束时间即可，现在时间可以用默认的参数，让js取得本地时间，不过在对时间要求非常精确的时候，建议在php里取得服务器时间作为现在时间。如果倒计时的开始时间在现在时间之后，那么就必须在参数里规定开始时间。

参数 默认值 功能
start null 开始时间
end new Date(Date.UTC(2050, 0, 1)) 结束时间
now new Date().getTime() 现在时间
mode 1 模式：默认模式1，显示天、小时、分、秒，模式2，显示小时、分、秒，模式3，显示小时、分、秒，其中秒显示为一位浮点小数
cls {day: 'ct-day',hour: 'ct-hour',min: 'ct-min',sec: 'ct-sec',dsec: 'ct-dsec'} 规定了数字部分的class名
startHtml '尚未开始' 未开始时显示的文字
endHtml '已结束' 结束后显示的文字
padding true 如果这个参数为true，那么数字小于十的时候，则自动补上0，例如03、04
useHtml false 如果为true，则不显示倒计时时间，只会显示未开始或者已结束。

代码

<span class="ct-day">05</span>天 <span class="ct-hour">01</span>小时 <span class="ct-min">09</span>分 <span class="ct-sec">51</span>秒
<script type="text/javascript">
$("#tcd-example1").timeCountDown({start: new Date("August 8, 2011 23:59:59"),end: new Date("August 12, 2012 23:59:59")});
</script>
