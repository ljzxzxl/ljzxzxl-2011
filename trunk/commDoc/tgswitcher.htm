tabSwitcher

功能：标签切换，可以定义触发方式，切换时间，自动播放，切换效果。

参数 默认值 功能
startIndex: 0 起始位置
activeCls 'cur' 选中状态的标签class
tabCls '.tab' 标签的class名
itemCls '.tab-item' 标签页的class名
delay 0 标签页切换的时间
duration 5000 自动切换的时间间隔
autoPlay: false 是否自动切换 true or false
trigger "click" 触发方式 建议 click、 over
omitLinks false 暂无
effect false 暂时只有slide
  • 最简单的tab
  • 自动切换的tab
  • 鼠标移上切换的tab
  • 有切换效果的tab

例子——最简单的tab

说明

  • 首先建立一个div#tabswitch-example1作为总的容器。
  • 然后在div#tabswitch-example下建立一个ul，li的class设为。
  • 接着在建立ul，li的class设为tab-item。
  • 设定一下.table的样式，以及选中状态的.cur样式。
  • 最后将tabSwitcher绑定在#tabswitch-example1上即可。

代码

.tab{ background-color: #FFFFFF;border-right: 1px solid #CCCCCC;float: left;text-align: center;width: 60px;}
.cur{ background-color:#000; color:#fff;}
<div id="tabswitch-example1">
<ul>
<li class="tab">1</li>
<li class="tab">2</li>
<li class="tab">3</li>
</ul>
<ul>
<li class="tab-item"><img src="images/s1.jpg" /></li>
<li class="tab-item"><img src="images/s2.jpg" /></li>
<li class="tab-item"><img src="images/s3.jpg" /></li>
</ul>
</div>
<script type="text/javascript">
$("#tabswitch-example1").tabSwitcher();
</script>

返回
