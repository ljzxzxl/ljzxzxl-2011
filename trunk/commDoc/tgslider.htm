tgSlider

功能：幻灯片效果，可以规定自动或者手动滚动，是否循环，方向，每次显示的个数等等。

参数 默认值 功能
btnPrevCls '.btn-prev' 向前滚动按钮的class名
btnNextCls '.btn-next' 向后滚动按钮的class名
pageCls '.ctl-page' 序号按钮的class名
containerCls '.slide-content' 滚动内容所在的容器class名
itemCls 'li' 滚动元素
activeCls 'cur' 序号按钮的选中状态class名，这里不用加“.”
perItem 5 滚动内容中显示的元素数
startIndex 0 从第几个元素开始
autoPlay false 是否自动滚动 true or false
duration 5000 自动播放时的时间间隔
fxDuration 600 滚动的速度
circle true 是否循环滚动 true or false
direction 'horizon' 方向，水平或者垂直 vertical or horizon
onShow 空 滚动时的函数
  • 最简单的自动滚动Slide
  • 垂直滚动的slide
  • 手动控制的Slide
  • 翻页键控制的Slide

例子——最简单的自动滚动Slide

说明

  • 首先建立一个div#slide-example作为总的容器，注意css中的position设为relative。
  • 然后在个div#slide-example下建立一个div.slide-content，css中的position设为relative，overflow设为hidden，宽度则根据slide元素的宽度和要显示的个数设定。
  • 接着把slide部分（这里是ul）放在div.slide-content下，注意position设为absolute。
  • ul下的li元素就是滚动的元素了，这是tgSlide插件中默认的item值，这里要把宽高设置一下,最好把float设置为left。
  • 最后就是将tgSlide绑定在div#slide-example上，并设定参数。

代码

.slide-banner { position: relative; height:106px; margin-left:10px;}
.slide-content { position: relative; width:256px; height:106px; overflow: hidden; }
.slide-content ul { position: absolute; }
.slide-content li { float: left; width: 256px; height: 106px; }
<div id="slide-example" class="slide-banner">
<div class="slide-content">
<ul>
<li><img src="images/s1.jpg" alt="" /></li>
<li><img src="images/s2.jpg" alt="" /></li>
<li><img src="images/s3.jpg" alt="" /></li>
<li><img src="images/s1.jpg" alt="" /></li>
<li><img src="images/s2.jpg" alt="" /></li>
<li><img src="images/s3.jpg" alt="" /></li>
</ul>
</div>
</div>
<script type="text/javascript">
$("#slide-example").tgSlider({
containerCls: 'ul',
perItem: 1,
autoPlay: true});
</script>

返回

例子——垂直滚动的slide

说明

这儿的css和html跟之前的例子是完全一样的，只要在tgSlide的参数里修改direction参数为vertical

<script type="text/javascript">
$("#slide-example").tgSlider({
containerCls: 'ul',
direction: 'vertical',
perItem: 1,
autoPlay: true});
</script>

返回

说明

需要增加翻页部分div.slide-ctls，并在参数中确认翻页部分为.slide-ctls span，并规定翻页元素被选中时的css。

.slide-ctls { position: absolute; top: 110px; left: 160px; width:100px; }
.slide-ctls li { float: left; margin: 0 5px 0 0; }
.slide-ctls span { background: #db660f; cursor:pointer; border: 1px solid #ae520d; font-family: sans-serif; font-weight: bold; font-size: 10px; float: left; width: 18px; height: 18px; line-height: 18px; text-align: center; color: #fff; }
.slide-ctls .cur { background: #fff; border: 1px solid #d16210; color: #db660f; }
<div id="slide-example4" class="slide-banner">
<div class="slide-content">
<ul>
<li><img src="images/s1.jpg" alt="" /></li>
<li><img src="images/s2.jpg" alt="" /></li>
<li><img src="images/s3.jpg" alt="" /></li>
<li><img src="images/s1.jpg" alt="" /></li>
</ul>
</div>
<div class="slide-ctls">

<li><span>2</span></li>
<li><span>3</span></li>
<li><span>4</span></li>
</ul>
</div>
</div>
<script type="text/javascript">
$("#slide-example4").tgSlider({
containerCls: 'ul',
direction: 'horizon',
pageCls:'.slide-ctls span',
perItem: 1});
</script>
